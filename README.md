### 使用方法

下载nodejs

使用如下命令安装cnpm

> npm install -g cnpm --registry=https://registry.npm.taobao.org

使用如下命令安装依赖

> cnpm install

执行complile_project来编译项目

执行run_test来直接运行

执行build_setup.bat可以打包win安装包
